# CMatrixOperations

Basic matrix structure and operations for the "Temas selectos de ecuaciones diferenciales ordinarias" course at ESFM-IPN

```c
void printMatrix(matrix M){
	for(i=0;i<5;i++){
		for(j=0;j<6;j++){
			printf("%lf ",M[i][j]);
		}
		printf("\n");
	}
}
```
